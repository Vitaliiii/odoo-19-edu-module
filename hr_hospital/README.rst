===========
HR Hospital
===========

Повноцінна система управління лікарнею для Odoo 19.

Опис
====
Цей модуль додає функціонал для ведення обліку лікарів, інтернів та пацієнтів, 
дозволяє створювати візити, ставити діагнози та формувати звіти.

Функції
=======
* Ієрархічний довідник хвороб.
* Розширені права доступу для Пацієнтів, Лікарів, Менеджерів та Адміністраторів.
* Локалізація (українська мова).
* Звіти у форматі PDF (Картка лікаря).
* Система менторства для інтернів.
